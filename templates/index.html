О сервисе

Рубрикатор реализован с использованием:

библиотеки для машинного обучения scikit-learn; поискового движка sphinx; парсера для синтаксического разбора HTML BeautifulSoup; библиотеки для анализа данных pandas

Использовался метод обучения с учителем - Метод опорных векторов (Support vector machines ) Корпус состоял из 25000 различных новостей 8 рубрик: Политика, Общество, Экономика, Происшествия, Спорт, Наука, Культура, Религия.
